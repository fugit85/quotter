FAQ

Can I upload a file from Keyword Planner?

Yes — the tool accepts CSV exports from Google Keyword Planner as-is. Keywords are read automatically from the Keyword column.

Does it handle large lists?

No hard limit — paste 10,000+ lines; processing runs in your browser and is not sent to our servers.

Is letter case preserved?

By default, case is unchanged. Under Advanced options you can normalize case — for example Title Case for ad headlines.

Can I wrap each word separately?

Yes — under Advanced options choose Words. The text is split into tokens and each is wrapped. Handy for single-word negatives or exact-match terms.

Can I remove duplicates before wrapping?

Yes — enable Remove under Duplicate lines. Lines with the same words in different order count as duplicates (e.g. “buy sofa” and “sofa buy”).

What does “Strip” special characters do?

It removes punctuation and symbols from each unit before wrapping. Letters and digits stay. Useful when the list was copied from a spreadsheet with extra characters.
