FAQ

Can I upload a file from Google Keyword Planner?

Yes — the tool accepts CSV directly from Google Keyword Planner without any editing. Keywords are extracted automatically from the Keyword column.

Does it handle large keyword lists?

No limits — paste 10,000+ lines, processing is instant and happens entirely in the browser without sending data to a server.

Will the keyword case be preserved?

Yes, by default the case is not changed. In Advanced options you can convert to any case — for example Title Case for ad headlines.

Can I wrap each word separately?

Yes — in Advanced options select "Each word". The tool will split the entire text into individual words and wrap each one. Useful for preparing single-word negative keywords or exact match terms.

Can I remove duplicates before wrapping?

Yes — in Advanced options enable "Remove duplicates". The tool also treats lines with the same words in different order as duplicates — "buy sofa" and "sofa buy" will be considered the same keyword.

What does "Remove special characters" do?

It removes punctuation and special characters (!, ?, @, #, % etc.) from each line before wrapping. Numbers and letters are kept. Useful when your list was copied from a spreadsheet or document with extra symbols.
