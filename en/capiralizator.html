FAQ

What is the difference between “First letter” and “Every letter”?

First letter of each word — title case per token: “buy a red sofa” → “Buy A Red Sofa”. Every letter uppercase — ALL CAPS: “buy sofa” → “BUY SOFA”.

Why use this for Google Ads?

Title case in headlines is a common CTR pattern. This tool normalizes long lists without manual editing.

Does it handle large lists?

Yes — paste as many lines as you need; everything runs in the browser.

Is line structure preserved?

Yes. Only letter case changes; line breaks stay in place.
